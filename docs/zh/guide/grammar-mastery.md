# 语法掌握

## 高频痛点

- 时态一致与完成时
- 条件与虚拟（If / Were / Had + done）
- 定语/名词/状语从句
- 非谓语（动名词/分词/不定式）
- 主被动与介词搭配

## 训练循环（15—25 分钟）

1. 例句：选 3—5 个高质量例句（来自权威语料）
2. 变式：改主语/时态/语态，写 5—8 个变式
3. 产出：结合自身话题写 5—8 句
4. 反馈：工具改 + 语料对照 + 人工互改

## 小测清单

- 时态线清晰（过去/现在/将来/完成）
- 从句连接词与指代正确
- 非谓语逻辑主语明确
- 搭配自然（介词/固定搭配）

## 推荐资料

- Practical English Usage（Swan）
- Grammar in Use（Cambridge）
- YouGlish/Skell：检索真实搭配
