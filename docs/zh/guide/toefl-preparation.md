# 托福备考

## 分项策略

- 听力：学术讲座精听 + 记笔记符号体系
- 口语：独立与综合题模版 + 录音自评
- 阅读：段落主旨与题干定位 + 干扰项识别
- 写作：综合阅读听力笔记 → 结构化输出

## 资料

- 官方 TPO；Noteful/ETS 官方指南
- 词汇：Academic Word List（AWL）

## 8 周计划（示例）

- 周 1—2：题型熟悉 + 基础能力（影子跟读/精读）
- 周 3—6：每日 1 题口语 + 每周 2 篇写作 + 2 套综合练习
- 周 7—8：全真模考 2—3 套 + 纠错与微调
