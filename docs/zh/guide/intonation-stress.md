# 语音语调

## 核心要点

- 意群（Thought Group）：逗号/停顿处切分
- 句重音（Sentence Stress）：信息重点落在实词上
- 语调（Intonation）：陈述/疑问/列举/对比的升降调

## 训练法

1. 标注：在文本上划分意群与重音
2. 模仿：对齐原音的节奏与重音位置
3. 表达：同一内容尝试不同语调表达态度

## 任务（10—15 分钟）

- 一段 3—5 句的材料，做 3 次不同语气的朗读
- 录音对比波形/时长，关注停顿与重音
