# 语法练习

## 高频专题

- 时态与体：一般/进行/完成/完成进行
- 从句：定语/名词/状语从句与连接词
- 非谓语：to do / doing / done 的用法
- 语态：主动 vs 被动的信息焦点

## 练习模版

1. 仿写：根据例句写 5 个变式
2. 选择：在 A/B 中选更自然的句子并说明理由
3. 改写：主动↔被动；简单句 ↔ 并从复合句

## 自测清单

- 时态一致性；代词与指代清晰
- 介词与搭配正确；不多余不缺少
