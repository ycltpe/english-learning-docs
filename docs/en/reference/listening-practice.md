# Listening Practice

## Levels

- Level 1: VOA/BBC6 with transcripts, 1–2 min
- Level 2: TED-Ed/ESLPod, 2–4 min
- Level 3: news/podcast segments, 3–6 min

## Checklist

- Dictate 5–8 key sentences
- Shadow 10× (segments → full)
- Retell 1–2× (record)

## Sources

- BBC 6 Minute English / VOA Learning English
- TED-Ed
