# Learning Apps

- Vocabulary: Anki, Quizlet, Rememberry (Chrome)
- Listening: podcast apps, YouTube + transcripts
- Speaking: italki, HelloTalk, ELSA Speak
- Writing: Grammarly, Languagetool, Write & Improve
- Corpora: YouGlish, Skell, Linguee
